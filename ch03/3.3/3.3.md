# 3.3 节的练习

### 3.3.1

对于下列各个语言

- C
- C++
- C#
- Fortran
- Java
- Lisp
- SQL

查询语言使用手册以确定：

1. 形成各语言的输入字母表的字符集分别是什么（不包括哪些只能出现在字符串或注释中的字符）？
2. 各语言的数字常量的语法形式是什么？
3. 各语言的标识符的词法形式是什么？
